```shell
    yarn json-server server.json -d 1000 -w 
```

### Para funcionar no android
```shell
    adb reverse tcp:3000 tcp:3000
```
ou 
```shell
    ~/Android/Sdk/platform-tools/adb reverse tcp:3000 tcp:3000
```
